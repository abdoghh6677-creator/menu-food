# نظام المخزون الذكي

## نظرة عامة

نظام المخزون الذكي مصمم للمطاعم لتتبع المخزون وإدارته بكفاءة مع خصم تلقائي من المخزون عند قبول الطلبات.

## المميزات

### ✅ الخصم التلقائي من المخزون
- خصم تلقائي عند قبول الطلبات
- منع البيع عند نفاد المخزون
- تحديث فوري لحالة المخزون

### ✅ تنبيهات ذكية
- تنبيهات انخفاض المخزون
- إشعارات عند الوصول للحد الأدنى
- تقارير دورية عن حالة المخزون

### ✅ إدارة شاملة
- تتبع التكاليف والموردين
- دعم وحدات مختلفة (قطعة، كيلو، لتر، إلخ)
- تحديثات جماعية للمخزون
- واجهة سهلة الاستخدام

## التثبيت

1. شغّل ملف `database/smart_inventory.sql` في Supabase SQL Editor
2. أعد تشغيل التطبيق
3. ستجد قسم "المخزون" في لوحة التحكم

## كيفية الاستخدام

### 1. تفعيل تتبع المخزون للعناصر

في صفحة المنيو:
- اضغط على "تعديل" لأي عنصر
- فعّل خيار "تتبع المخزون"
- اضبط المخزون الأولي

### 2. إدارة المخزون

في صفحة المخزون:
- عرض جميع عناصر المخزون
- إضافة عناصر جديدة
- تعديل المخزون والحدود
- تحديث جماعي للمخزون

### 3. الخصم التلقائي

عند قبول طلب:
- يتم خصم الكميات تلقائياً
- تحديث فوري للمخزون
- تنبيه عند انخفاض المخزون

## الإحصائيات المتاحة

- إجمالي عدد العناصر في المخزون
- عدد العناصر ذات المخزون المنخفض
- عدد العناصر التي نفد مخزونها
- إجمالي قيمة المخزون

## التنبيهات

### تنبيهات انخفاض المخزون
تظهر تلقائياً عندما يصل المخزون للحد الأدنى المحدد.

### تنبيهات نفاد المخزون
تحذير عندما يصل المخزون للصفر.

## الوحدات المدعومة

- قطعة
- كيلو
- لتر
- علبة
- حزمة

## API المتاح

### وظائف المخزون في `restaurantService.ts`

```typescript
// الحصول على عناصر المخزون
getInventoryItems(restaurantId: string): Promise<InventoryItem[]>

// إضافة عنصر مخزون
createInventoryItem(item: Partial<InventoryItem>): Promise<boolean>

// تحديث عنصر مخزون
updateInventoryItem(id: string, updates: Partial<InventoryItem>): Promise<boolean>

// حذف عنصر مخزون
deleteInventoryItem(id: string): Promise<boolean>

// تحديث جماعي
bulkUpdateInventory(restaurantId: string, updates: any[]): Promise<boolean>

// الحصول على تنبيهات انخفاض المخزون
getLowStockAlerts(restaurantId: string): Promise<any[]>

// الحصول على إحصائيات المخزون
getInventoryStats(restaurantId: string): Promise<object>
```

## قاعدة البيانات

### جدول `inventory`

| العمود | النوع | الوصف |
|--------|------|-------|
| id | UUID | المعرف الفريد |
| restaurant_id | UUID | معرف المطعم |
| menu_item_id | UUID | معرف عنصر المنيو |
| item_name | TEXT | اسم العنصر |
| current_stock | DECIMAL | المخزون الحالي |
| minimum_stock | DECIMAL | الحد الأدنى |
| unit | TEXT | الوحدة |
| cost_price | DECIMAL | سعر الشراء |
| supplier_name | TEXT | اسم المورد |
| auto_deduct | BOOLEAN | الخصم التلقائي |
| low_stock_alert | BOOLEAN | تنبيه انخفاض المخزون |
